# First Website at TechCrusades

This is my first-ever frontend project completed during my internship at TechCrusades. The project is built using only HTML and CSS after learning from W3Schools.

## About the Project

This project was created as part of my internship where strict guidelines were given not to use any AI tools or code generators. I was required to build everything manually after learning HTML and CSS concepts from W3Schools and practicing them on my own.

The purpose of this project was to understand the fundamentals of web development and how to structure and style a complete static website from scratch.

## Features

- Responsive layout using CSS media queries  
- Hero section with background image and overlay  
- Service cards layout using Flexbox  
- Contact and About sections  
- Clean footer section  
- Mobile-friendly design  

## Tech Stack

- HTML5  
- CSS3 (Flexbox, Media Queries)

## Learning Outcomes

- Understanding semantic HTML structure  
- CSS layout techniques using Flexbox  
- Responsive design with media queries  
- Working with images and background styling  
- Building a complete static website from scratch  

## How to Run This Project

1. Clone or download the repository  
2. Open the project folder  
3. Open `index.html` in any web browser  

## Note

This project was completed during my internship at TechCrusades under strict guidelines where external AI tools were not allowed. The entire project was developed through self-learning and practice using HTML and CSS from W3Schools.
